# Pune Hometown Homepage

A simple responsive homepage showcasing the city of Pune, built using HTML and CSS.

Live link: https://hometown-visit-sample-page.vercel.app
## Features

- Hero section with a background image
- Activities section highlighting popular attractions in Pune
- Tour guide card section
- Flexbox-based layouts
- Responsive image handling

## Technologies Used

- HTML5
- CSS3
- Flexbox
